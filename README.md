# የአማርኛ ቃላት መፍቻ - Amharic Smart Unscrambler 🇪🇹

A sleek, lightweight, single-page web application built to unscramble Amharic words using both **Exact Fidel Mapping** and **Phonetic Sound Analysis**.

## 🚀 Live Demo
[👉 Click here to open the app!](https://samkebz-y.github.io/Amharic--Unscrambler-/)

## ✨ Key Features
* **Exact Fidel Unscrambler:** Rearranges precise Amharic characters (`ምሰላ` ➡️ `ሰላም`).
* **Smart Phonetic Matcher:** Recognizes root consonant patterns even if the vowel endings are mixed up (`ስላመ` ➡️ `ሰላም`).
* **Mobile Optimized:** Clean dark-mode dashboard tailored for smooth smartphone usage.

## 🛠️ Built With
* HTML5
* CSS3 (Custom Dark Theme variables)
* Pure JavaScript (No external server APIs needed)
